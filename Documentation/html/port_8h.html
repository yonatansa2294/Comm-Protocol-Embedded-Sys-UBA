◆ config_Default_SIM()

bool_t config_Default_SIM ( )

Set the default initialization parameters for the USART as well as for the reset and powerkey pins.

Note
The default parameters are as follows: USART : USART1, Baud rate:9600, Bits:8, Parity: None, Stop bits: 1 PowerKey Pin: PIN_B0, initial state: OFF Reset Pin : PIN_B1, initial state: ON In addition, the SIM remains turn off and reset until the power_On_SIM function is used. This is because some modules do not have the powerKey pin, so we reset the SIM.
Parameters
None.
Return values
Initializationstatus. If the initialization was successful it returns true, otherwise it returns false.

◆ power_Down_SIM()

void power_Down_SIM ( )

Turn off the SIM.

Note
The start-up sequence is as follows:
  1. Hold the powerKey pin high for at least 1.5s and then set it low. This is specified in the Hardware Design Guide v1.9, page 23.
  2. Wait a time of at least 3s for the SIM UART to to wait for the UART SIM goes to the idle state.
Parameters
None.
Return values
None.

◆ power_On_SIM()

void power_On_SIM ( )

Turn on the SIM.

Note
The start-up sequence is as follows:
  1. Hold the powerKey pin high for a time of at least 1s and then set it low. This is specified in the Hardware Design Guide v1.9, page 22.
  2. Wait a time of at least 15s for the SIM UART to be released. On power-up, the SIM sends some initial status messages over the UART, which are not relevant, so we wait for this time.
Parameters
None.
Return values
None.
